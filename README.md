# Super Intelligent Binance Futures Trading Bot

Advanced auto-trading bot featuring LSTM predictions, PPO reinforcement learning, real-time WebSocket, multi-symbol portfolio, and sentiment analysis.

## Features
- Real-time data via WebSocket
- LSTM for price forecasting
- PPO for adaptive trading policy
- Periodic model retraining
- Market sentiment integration
- Multi-symbol support (BTC, ETH, etc.)

## Quick Start
1. `pip install -r requirements.txt`
2. Copy `.env.example` to `.env` and fill API keys
3. Run on **Testnet** first: `python super_trader_enhanced.py`

**⚠️ High Risk**: Trading involves substantial risk of loss. Backtest thoroughly.
